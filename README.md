# E-Commerce Conversion Funnel Analysis

## Project Overview
This project analyzes user behavior in an e-commerce platform using Python.

## Objectives
- Analyze Visit → View → Cart → Purchase funnel
- Calculate conversion rates
- Identify user drop-offs
- Analyze top brands and categories
- Visualize results using charts

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset
E-commerce Behavior Dataset (Kaggle)
